# Landings de productos

El sitio ya esta preparado para agregar landings futuras en Astro.

## Opcion rapida con Markdown

1. Duplica `src/content/productos/ejemplo.md.example`.
2. Cambia la extension a `.md`.
3. Actualiza titulo, descripcion, caracteristicas y contenido.
4. La pagina se generara en `/productos/nombre-del-archivo/`.

## Opcion con codigo Astro

Duplica `src/pages/productos/_plantilla.astro`, renombrala como `nombre-del-producto.astro`
y cambia el contenido. Esta opcion sirve cuando la landing necesita secciones mas personalizadas.
